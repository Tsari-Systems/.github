> **A knowing institution is one whose reasoning outlasts the people who did it.**

Most organizations employ knowers. Almost none are *knowing*.

Every AI-augmented session ends. The output survives. The reasoning chain that
produced it — the sources, the warrant, the judgment — does not. What institutions
are currently inheriting is conclusions stripped of everything that made them
credible. Tsari builds the infrastructure that changes this: warranted records,
traceable decisions, and reasoning that future humans and systems can inherit.

Tsari is an **independent research and product institution** — not a productivity
tool, not a memory product, not an AI assistant.

## The object lifecycle

The core primitive. Every piece of knowledge earns its authority at each stage
before it can be inherited:

**Observation → Finding → Claim → Ratified → Inherited**

A claim without a traceable source is not yet a claim. Provenance is not a
feature — it is the definition of knowing.
